# Compra coletiva de café

Aplicação web inicial para gestão de pedidos coletivos de café.

## Status atual
- interface de login e painel principal funcionando localmente;
- criação de pedidos e participação com cálculo de valor por kg;
- confirmação de retirada por parte do usuário;
- armazenamento local em navegador para demonstração do fluxo MVP.

## Próximo passo
Substituir o armazenamento local por Firebase Authentication + Firestore para persistência real entre usuários.
